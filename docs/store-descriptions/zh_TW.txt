【🚀 v2 新功能】:
    • 🚀 側邊欄模式 - 頁面旁始終可見的分頁樹（Alt + S）
    • 🚀 分頁群組 - 完整支援原生分頁群組，帶有顏色編碼容器、與瀏覽器同步的摺疊/展開，以及內嵌編輯（雙擊重新命名和選擇顏色）
    • 🚀 工作區 - 將目前視窗儲存為命名工作區。稍後還原所有分頁、樹狀結構、群組和標記
    • 🚀 分頁標記 - 使用圖示標記分頁（✓ 完成、📌 釘選、✗ 拒絕、⚠ 進行中、? 疑問）以便快速視覺掃描
    • 🚀 搜尋與篩選 - 帶關鍵字醒目顯示的即時篩選。書籤也會出現在搜尋結果中
    • 🚀 10種語言 - 英語、德語、西班牙語、法語、日語、韓語、葡萄牙語（巴西）、俄語、簡體中文、繁體中文

【兩種模式】:
    • 彈出模式（Alt + Q） - 快速切換分頁的彈出覆蓋層，操作後自動關閉
    • 側邊欄模式（Alt + S） - 始終顯示在頁面旁邊，適合管理大量分頁

【鍵盤快捷鍵】:
    • Alt + Q：開啟彈出覆蓋層
    • Alt + S：開啟側邊欄
    • ↑（上箭頭）：移動到上一個分頁
    • ↓（下箭頭）：移動到下一個分頁
    • ←（左箭頭）：摺疊節點或前往父節點
    • →（右箭頭）：展開節點或前往第一個子節點
    • ⏎（Enter）：切換到選取的分頁 / 開啟書籤 / 網路搜尋
    • Alt + W：關閉選取的分頁及其所有子分頁
    • 在任意位置輸入即可即時搜尋和篩選

【簡介】:
    Tree Style Tab 解決了一個簡單但令人煩惱的問題：當你開啟了數十個分頁時，如何快速找到並切換到你需要的分頁？

    與其他分頁管理工具不同，Tree Style Tab 以樹狀結構顯示你的分頁。分頁按其父子關係自動組織——從某個分頁開啟的任何分頁都會成為它的子節點。這為你提供了瀏覽情境的清晰層級視圖。

    透過拖放操作，自由重組你的分頁樹——在群組間移動分頁、重新排序同級分頁，或巢狀為子節點。即時指示器精確顯示分頁將放置的位置。

【功能】:
    • 樹狀檢視 - 自動的父子樹狀結構，帶摺疊/展開和 +N 徽章
    • 分頁群組 - 原生瀏覽器分頁群組，9種顏色，同步摺疊，內嵌編輯
    • 拖放 - 在樹中任意位置重新排列分頁，甚至跨群組
    • 搜尋 - 按標題或 URL 即時篩選分頁；包含書籤（最多30個結果）
    • 網路搜尋回退 - 沒有結果？按 Enter 直接在網路上搜尋
    • 工作區 - 儲存、預覽、編輯和還原分頁工作階段（側邊欄）
    • 分頁標記 - 5種標記類型，帶彩色網站圖示徽章（側邊欄）
    • 深色/淺色主題 - 自動適應系統偏好設定
    • 視覺指示器 - 網站圖示、載入動畫、使用中分頁醒目顯示
    • 摺疊徽章 - 分支摺疊時顯示隱藏的子節點數量

【注意】:
    在安裝此擴充功能之前開啟的分頁將顯示為獨立的根節點（無法偵測已有分頁的父子關係）。
