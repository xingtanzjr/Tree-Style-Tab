【🚀 v2 새로운 기능】:
    • 🚀 사이드 패널 모드 - 페이지 옆에 항상 표시되는 탭 트리 (Alt + S)
    • 🚀 탭 그룹 - 색상 코드 컨테이너, 브라우저와 동기화된 접기/펼치기, 인라인 편집(더블 클릭으로 이름 변경 및 색상 선택)을 포함한 네이티브 탭 그룹 완전 지원
    • 🚀 워크스페이스 - 현재 창을 이름이 있는 워크스페이스로 저장. 모든 탭, 트리 구조, 그룹 및 마크를 나중에 복원
    • 🚀 탭 마크 - 아이콘으로 탭 마크 (✓ 완료, 📌 고정, ✗ 거부, ⚠ 진행 중, ? 질문) 빠른 시각적 스캔
    • 🚀 검색 & 필터 - 키워드 하이라이트가 포함된 실시간 필터링. 북마크도 결과에 표시
    • 🚀 10개 언어 - 영어, 독일어, 스페인어, 프랑스어, 일본어, 한국어, 포르투갈어(브라질), 러시아어, 중국어 간체, 중국어 번체

【두 가지 모드】:
    • 팝업 모드 (Alt + Q) - 빠른 탭 전환을 위한 퀵 오버레이, 동작 후 닫힘
    • 사이드 패널 모드 (Alt + S) - 페이지 옆에 항상 표시, 많은 탭 관리에 이상적

【키보드 단축키】:
    • Alt + Q : 팝업 오버레이 열기
    • Alt + S : 사이드 패널 열기
    • ↑ (위쪽 화살표) : 이전 탭으로 이동
    • ↓ (아래쪽 화살표) : 다음 탭으로 이동
    • ← (왼쪽 화살표) : 노드 접기 또는 부모로 이동
    • → (오른쪽 화살표) : 노드 펼치기 또는 첫 번째 자식으로 이동
    • ⏎ (Enter) : 선택한 탭으로 전환 / 북마크 열기 / 웹 검색
    • Alt + W : 선택한 탭과 모든 하위 탭 닫기
    • 어디서든 입력하여 즉시 검색 및 필터링

【소개】:
    Tree Style Tab은 간단하지만 짜증나는 문제를 해결합니다: 수십 개의 탭이 열려 있을 때 필요한 탭을 어떻게 빨리 찾고 전환할 수 있을까요?

    다른 탭 관리자와 달리 Tree Style Tab은 탭을 트리 구조로 표시합니다. 탭은 부모-자식 관계에 따라 자동으로 정리됩니다 — 다른 탭에서 열린 탭은 그 탭의 자식이 됩니다. 이를 통해 브라우징 컨텍스트의 명확한 계층적 뷰를 얻을 수 있습니다.

    드래그 앤 드롭으로 탭 트리를 자유롭게 재구성하세요 — 그룹 간 탭 이동, 형제 재정렬 또는 자식으로 중첩. 라이브 인디케이터가 탭이 놓일 위치를 정확히 보여줍니다.

【기능】:
    • 트리 뷰 - 접기/펼치기 및 +N 배지가 있는 자동 부모-자식 트리
    • 탭 그룹 - 9가지 색상의 네이티브 브라우저 탭 그룹, 동기화된 접기, 인라인 편집
    • 드래그 앤 드롭 - 트리의 어디에서나 탭 재배치, 그룹 간도 가능
    • 검색 - 제목 또는 URL로 즉시 탭 필터링; 북마크 포함 (최대 30개 결과)
    • 웹 검색 대체 - 결과 없음? Enter를 눌러 웹에서 직접 검색
    • 워크스페이스 - 탭 세션 저장, 미리보기, 편집 및 복원 (사이드 패널)
    • 탭 마크 - 색상 파비콘 배지가 있는 5가지 마크 유형 (사이드 패널)
    • 다크 / 라이트 테마 - 시스템 설정에 자동 적응
    • 시각적 인디케이터 - 파비콘, 로딩 스피너, 활성 탭 하이라이트
    • 접기 배지 - 브랜치가 접혀 있을 때 숨겨진 자식 수 표시

【알림】:
    이 확장 프로그램을 설치하기 전에 열린 탭은 별도의 루트 노드로 표시됩니다 (기존 탭에 대해서는 부모-자식 관계를 감지할 수 없습니다).
