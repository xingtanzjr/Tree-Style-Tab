【🚀 v2の新機能】:
    • 🚀 サイドパネルモード - ページの横に常に表示されるタブツリー（Alt + S）
    • 🚀 タブグループ - ネイティブタブグループの完全サポート。色分けされたコンテナ、ブラウザと同期した折りたたみ/展開、インライン編集（ダブルクリックで名前変更と色選択）
    • 🚀 ワークスペース - 現在のウィンドウを名前付きワークスペースとして保存。すべてのタブ、ツリー構造、グループ、マークを後で復元
    • 🚀 タブマーク - アイコンでタブをマーク（✓ 完了、📌 ピン、✗ 却下、⚠ 作業中、? 質問）で素早く視覚的にスキャン
    • 🚀 検索＆フィルター - キーワードハイライト付きのリアルタイムフィルタリング。ブックマークも結果に表示
    • 🚀 10言語対応 - 英語、ドイツ語、スペイン語、フランス語、日本語、韓国語、ポルトガル語（ブラジル）、ロシア語、簡体字中国語、繁体字中国語

【2つのモード】:
    • ポップアップモード（Alt + Q） - タブの素早い切り替え用のクイックオーバーレイ、アクション後に閉じる
    • サイドパネルモード（Alt + S） - ページの横に常に表示、多くのタブの管理に最適

【キーボードショートカット】:
    • Alt + Q : ポップアップオーバーレイを開く
    • Alt + S : サイドパネルを開く
    • ↑（上矢印） : 前のタブに移動
    • ↓（下矢印） : 次のタブに移動
    • ←（左矢印） : ノードを折りたたむまたは親に移動
    • →（右矢印） : ノードを展開または最初の子に移動
    • ⏎（Enter） : 選択したタブに切り替え / ブックマークを開く / ウェブ検索
    • Alt + W : 選択したタブとすべての子タブを閉じる
    • どこでも入力して即座に検索・フィルタリング

【紹介】:
    Tree Style Tabはシンプルだけど厄介な問題を解決します：何十ものタブが開いている時、必要なタブをどうやって素早く見つけて切り替えるか？

    他のタブマネージャーとは異なり、Tree Style Tabはタブをツリー構造で表示します。タブは親子関係によって自動的に整理されます — あるタブから開かれたタブはその子になります。これにより、ブラウジングコンテキストの明確な階層ビューが得られます。

    ドラッグ＆ドロップで、タブツリーを自由に再編成 — グループ間でタブを移動、兄弟を並び替え、または子としてネスト。ライブインジケーターがタブの着地点を正確に表示します。

【機能】:
    • ツリービュー - 折りたたみ/展開と+Nバッジ付きの自動親子ツリー
    • タブグループ - 9色のネイティブブラウザタブグループ、同期された折りたたみ、インライン編集
    • ドラッグ＆ドロップ - ツリーのどこでもタブを再配置、グループ間も可能
    • 検索 - タイトルまたはURLで即座にタブをフィルタリング、ブックマークも含む（最大30件）
    • ウェブ検索フォールバック - 結果なし？Enterキーでウェブを直接検索
    • ワークスペース - タブセッションの保存、プレビュー、編集、復元（サイドパネル）
    • タブマーク - 色付きファビコンバッジ付き5種類のマーク（サイドパネル）
    • ダーク/ライトテーマ - システム設定に自動適応
    • 視覚インジケーター - ファビコン、ローディングスピナー、アクティブタブのハイライト
    • 折りたたみバッジ - ブランチが折りたたまれた時に非表示の子の数を表示

【注意】:
    この拡張機能をインストールする前に開かれたタブは、独立したルートノードとして表示されます（既存のタブに対しては親子関係を検出できません）。
