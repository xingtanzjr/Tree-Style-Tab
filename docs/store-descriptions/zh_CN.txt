【🚀 v2 新功能】:
    • 🚀 侧边栏模式 - 页面旁始终可见的标签页树（Alt + S）
    • 🚀 标签页分组 - 完整支持原生标签页分组，带有颜色编码容器、与浏览器同步的折叠/展开，以及内联编辑（双击重命名和选择颜色）
    • 🚀 工作区 - 将当前窗口保存为命名工作区。稍后恢复所有标签页、树结构、分组和标记
    • 🚀 标签页标记 - 使用图标标记标签页（✓ 完成、📌 固定、✗ 拒绝、⚠ 进行中、? 疑问）以便快速视觉扫描
    • 🚀 搜索与筛选 - 带关键词高亮的实时筛选。书签也会出现在搜索结果中
    • 🚀 10种语言 - 英语、德语、西班牙语、法语、日语、韩语、葡萄牙语（巴西）、俄语、简体中文、繁体中文

【两种模式】:
    • 弹出模式（Alt + Q） - 快速切换标签页的弹出覆盖层，操作后自动关闭
    • 侧边栏模式（Alt + S） - 始终显示在页面旁边，适合管理大量标签页

【键盘快捷键】:
    • Alt + Q：打开弹出覆盖层
    • Alt + S：打开侧边栏
    • ↑（上箭头）：移动到上一个标签页
    • ↓（下箭头）：移动到下一个标签页
    • ←（左箭头）：折叠节点或转到父节点
    • →（右箭头）：展开节点或转到第一个子节点
    • ⏎（回车）：切换到选中的标签页 / 打开书签 / 网络搜索
    • Alt + W：关闭选中的标签页及其所有子标签页
    • 在任意位置输入即可即时搜索和筛选

【简介】:
    Tree Style Tab 解决了一个简单但令人烦恼的问题：当你打开了数十个标签页时，如何快速找到并切换到你需要的标签页？

    与其他标签页管理器不同，Tree Style Tab 以树形结构显示你的标签页。标签页按其父子关系自动组织——从某个标签页打开的任何标签页都会成为它的子节点。这为你提供了浏览上下文的清晰层级视图。

    通过拖放操作，自由重组你的标签页树——在分组间移动标签页、重新排序同级标签页，或嵌套为子节点。实时指示器精确显示标签页将放置的位置。

【功能】:
    • 树形视图 - 自动的父子树形结构，带折叠/展开和 +N 徽章
    • 标签页分组 - 原生浏览器标签页分组，9种颜色，同步折叠，内联编辑
    • 拖放 - 在树中任意位置重新排列标签页，甚至跨分组
    • 搜索 - 按标题或 URL 即时筛选标签页；包含书签（最多30个结果）
    • 网络搜索回退 - 没有结果？按回车直接在网络上搜索
    • 工作区 - 保存、预览、编辑和恢复标签页会话（侧边栏）
    • 标签页标记 - 5种标记类型，带彩色网站图标徽章（侧边栏）
    • 深色/浅色主题 - 自动适应系统偏好设置
    • 视觉指示器 - 网站图标、加载动画、活动标签页高亮
    • 折叠徽章 - 分支折叠时显示隐藏的子节点数量

【注意】:
    在安装此扩展之前打开的标签页将显示为独立的根节点（无法检测已有标签页的父子关系）。
